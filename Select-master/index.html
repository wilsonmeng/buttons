默认样式：

$("#defaultSelect").Lx_SelectInit();

之定义样式：

$("#customSelect").Lx_SelectInit({
	width : 200,//下拉框固定宽度
	height : 35,//下拉框高度也是行高
	maxWidth : 300,//下拉框下拉选项最大宽度
	optionMinNum : 2,//下拉框下拉选项最小展示条数
	optionMaxNum : 6,//下拉框下拉选项最大展示条数
	zIndex:100
});

改变select：

$("#select").click(function(){
	$("#changeSelect").find("option:eq(3)").prop("selected",true).change();//选中第三个选项
});
$("#disabled").click(function(){
	$("#changeSelect").find("option:eq(4)").prop("disabled",true).change();//禁用第四个选项
});
